
# Band Name Generator

Welcome to the Band Name Generator project! This simple Python program generates unique and interesting band names based on user input. Whether you're forming a new band or just having fun, this program can provide some creative inspiration.

## Author

This project was created by [Pratyush Kumar Jha](https://www.linkedin.com/in/pratyush-kumar-jha-b37a57235/).

## Features

- Generates unique band names based on user input.
- Simple and easy-to-use command-line interface.
- Provides a fun way to come up with creative names for bands or other projects.

## Installation

To run this program, you'll need to have Python installed on your machine. You can download Python from the official website: [python.org](https://www.python.org/).

1. Clone this repository to your local machine:

```sh
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```sh
cd Python-Programs/Project\ 1\ Band_Name_Generator
```

## Usage

To generate a band name, run the `band_name_generator.py` script:

```sh
python band_name_generator.py
```

Follow the on-screen prompts to enter the required information, and the program will generate a unique band name for you.

## Example

Here's an example of what the output might look like:

```sh
Welcome to the Band Name Generator.
What's the name of the city you grew up in?
> New York
What's the name of your pet?
> Buddy
Your band name could be: New York Buddy
```
