
# Name Card Using Flask Project

Welcome to the Name Card Using Flask Project repository. This project, authored by Pratyush Kumar Jha, provides a web-based tool to create and display digital name cards. It is designed to offer a simple and efficient way to manage and share contact information.

## Introduction

The Name Card Using Flask Project is a web-based tool that allows users to create and display digital name cards. This project utilizes a template taken from another site to provide a visually appealing and user-friendly interface.

## Features

- Create digital name cards
- Display name cards in a web browser
- User-friendly design and interface
- Easy to share contact information

## Installation

To get started with this project, follow the steps below:

1. Clone the repository:

```bash
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```bash
cd Python-Programs/Project\ 40\ Name_Card_using\ Flask
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

To create and display name cards, run the Flask application:

```bash
python app.py
```

Open your web browser and navigate to `http://127.0.0.1:5000` to start using the tool.

## How the Code is Made

The project is structured as follows:

- **Backend**: The server-side functionality is implemented using Flask, a lightweight web framework for Python.
- **Frontend**: The user interface is created using a template taken from another site, along with HTML, CSS, and JavaScript for customization and interactivity.
- **Routing**: Flask routes are used to manage the different pages and functionalities of the tool.
- **Error Handling**: The code includes error handling mechanisms to manage issues such as invalid inputs and server errors.
