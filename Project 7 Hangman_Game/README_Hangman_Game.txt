
# Hangman Game

Welcome to the Hangman Game project! This classic word-guessing game is implemented in Python and allows you to play against the computer. It's a fun way to practice your Python programming skills and enjoy a classic game.

## Author

This project was created by [Pratyush Kumar Jha](https://www.linkedin.com/in/pratyush-kumar-jha-b37a57235/).

## Features

- Play the classic Hangman game.
- Simple and interactive command-line interface.
- Random word selection for each game.
- Displays the number of attempts left and the current state of the word.

## Installation

To run this program, you'll need to have Python installed on your machine. You can download Python from the official website: [python.org](https://www.python.org/).

1. Clone this repository to your local machine:

```sh
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```sh
cd Python-Programs/Project\ 7\ Hangman_Game
```

## Usage

To start playing Hangman, run the `Project_7_Hangman_Game.py` script:

```sh
python Project_7_Hangman_Game.py
```

Follow the on-screen prompts to guess letters and try to complete the word before you run out of attempts.
