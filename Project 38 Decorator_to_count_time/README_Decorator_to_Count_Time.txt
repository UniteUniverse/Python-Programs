Decorator to Count Time
This project demonstrates how to use a Python decorator to measure and print the execution time of functions. The decorator wraps the target function and calculates the time taken for its execution.

Introduction
This script provides a simple example of using a decorator to measure the execution time of functions. It includes two functions, fast_function and slow_function, which are decorated to print their execution times.

Prerequisites
Before you begin, ensure you have met the following requirements:

Python installed on your system.
Installation
Clone this repository to your local machine:
git clone https://github.com/UniteUniverse/Python-Programs.git

Navigate to the project directory:
cd Python-Programs/Project%2038%20Decorator_to_count_time

Usage
Run the script:
python decorator_time_count.py

The script will execute the fast_function and slow_function, printing the time taken for each function to complete.
