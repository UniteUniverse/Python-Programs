
# Tip Calculator

Welcome to the Tip Calculator project! This simple Python program helps you calculate the amount each person needs to pay after adding a tip to the total bill. It's a handy tool for group dining situations to easily split the bill and include a tip.

## Author

This project was created by [Pratyush Kumar Jha](https://www.linkedin.com/in/pratyush-kumar-jha-b37a57235/).

## Features

- Calculates the total amount to be paid by each person.
- Includes a user-defined tip percentage.
- Simple and interactive command-line interface.

## Installation

To run this program, you'll need to have Python installed on your machine. You can download Python from the official website: [python.org](https://www.python.org/).

1. Clone this repository to your local machine:

```sh
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```sh
cd Python-Programs/Project\ 2\ Tip_Calculator
```

## Usage

To calculate the tip and total amount per person, run the `Project_2_Tip_Calculator.py` script:

```sh
python Project_2_Tip_Calculator.py
```

Follow the on-screen prompts to enter the required information, and the program will display the amount each person should pay.
