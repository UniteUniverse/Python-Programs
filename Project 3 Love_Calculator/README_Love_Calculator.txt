
# Love Calculator

Welcome to the Love Calculator project! This fun Python program calculates the compatibility percentage between two people based on their names. It's a playful way to see how well-matched you are with your significant other or crush.

## Author

This project was created by [Pratyush Kumar Jha](https://www.linkedin.com/in/pratyush-kumar-jha-b37a57235/).

## Features

- Calculates a compatibility percentage between two names.
- Simple and interactive command-line interface.
- Fun and entertaining for users.

## Installation

To run this program, you'll need to have Python installed on your machine. You can download Python from the official website: [python.org](https://www.python.org/).

1. Clone this repository to your local machine:

```sh
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```sh
cd Python-Programs/Project\ 3\ Love_Calculator
```

## Usage

To calculate the compatibility percentage, run the `Project_3_Love_Calculator.py` script:

```sh
python Project_3_Love_Calculator.py
```

Follow the on-screen prompts to enter the names, and the program will display the compatibility percentage.

## Example

Here's an example of what the output might look like:

```sh
Welcome to Love Calculator❤️
Enter your name: John
Enter your partner's name: Jane
The compatibility percentage between John and Jane is: 85%
```
