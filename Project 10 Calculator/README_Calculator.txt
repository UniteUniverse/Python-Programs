
# Calculator

Welcome to the Calculator project! This Python program implements a basic calculator that performs simple arithmetic operations like addition, subtraction, multiplication, and division. It's a great way to practice Python programming and understand basic arithmetic operations.

## Author

This project was created by [Pratyush Kumar Jha](https://www.linkedin.com/in/pratyush-kumar-jha-b37a57235/).

## Features

- Performs basic arithmetic operations: addition, subtraction, multiplication, and division.
- Simple and interactive command-line interface.
- Handles invalid inputs gracefully.

## Installation

To run this program, you'll need to have Python installed on your machine. You can download Python from the official website: [python.org](https://www.python.org/).

1. Clone this repository to your local machine:

```sh
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```sh
cd Python-Programs/Project\ 10\ Calculator
```

## Usage

To use the calculator, run the `Project_10_Calculator.py` script:

```sh
python Project_10_Calculator.py
```

Follow the on-screen prompts to perform arithmetic operations.

## Example

Here's an example of what the output might look like:

```sh
Welcome to the Calculator!
Enter the first number: 10
Enter the operation (+, -, *, /): +
Enter the second number: 5
The result is: 15
```
