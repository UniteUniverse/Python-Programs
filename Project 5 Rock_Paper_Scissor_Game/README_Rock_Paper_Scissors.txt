
# Rock Paper Scissors Game

Welcome to the Rock Paper Scissors Game project! This classic game is implemented in Python and allows you to play against the computer. It's a simple yet fun way to practice your Python programming skills.

## Author

This project was created by [Pratyush Kumar Jha](https://www.linkedin.com/in/pratyush-kumar-jha-b37a57235/).

## Features

- Play the classic Rock Paper Scissors game against the computer.
- Simple and interactive command-line interface.
- Randomized computer choices for a fair game.

## Installation

To run this program, you'll need to have Python installed on your machine. You can download Python from the official website: [python.org](https://www.python.org/).

1. Clone this repository to your local machine:

```sh
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```sh
cd Python-Programs/Project\ 5\ Rock_Paper_Scissor_Game
```

## Usage

To start playing Rock Paper Scissors, run the `Project_5_Rock_Paper_Scissor_Game.py` script:

```sh
python Project_5_Rock_Paper_Scissor_Game.py
```

Follow the on-screen prompts to make your choice and see if you can beat the computer.

## Example

Here's an example of what the output might look like:

```sh
Welcome to Rock Paper Scissors Game!
Choose one: rock, paper, scissors
> rock
You chose rock, computer chose scissors.
You win!
```
