Higher Lower Game Using Flask
This project is a simple web-based game where users guess a number between 0 and 9. The game provides feedback on whether the guess is too high, too low, or correct, using Flask for the web framework.

Introduction
This script creates a web application where users can play a number guessing game. The application generates a random number between 0 and 9 and provides feedback on the userâ€™s guess.

Prerequisites
Before you begin, ensure you have met the following requirements:

Python installed on your system.
Flask library installed. You can install it using pip:
pip install Flask

Installation
Clone this repository to your local machine:
git clone https://github.com/UniteUniverse/Python-Programs.git

Navigate to the project directory:
cd Python-Programs/Project%2039%20Higher_Lower_Game_Using_Flask

Install the required libraries:
pip install Flask

Usage
Run the script:
python higher_lower_game.py

Open your web browser and go to http://127.0.0.1:5000/.
Guess a number by appending it to the URL, for example: http://127.0.0.1:5000/5.
The application will provide feedback on whether your guess is too high, too low, or correct.
