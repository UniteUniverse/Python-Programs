
# Higher Lower Game Using Flask Project

Welcome to the Higher Lower Game Using Flask Project repository. This project, authored by Pratyush Kumar Jha, provides a web-based game where players guess if the next number will be higher or lower than the current one. It is designed to offer a simple and engaging gaming experience.

## Introduction

The Higher Lower Game Using Flask Project is a web-based game that challenges players to guess whether the next number in a sequence will be higher or lower than the current number. This game is built using the Flask framework, making it accessible via a web browser.

## Features

- Simple and intuitive gameplay
- Web-based interface accessible via a browser
- Tracks player scores
- User-friendly design

## Installation

To get started with this project, follow the steps below:

1. Clone the repository:

```bash
git clone https://github.com/UniteUniverse/Python-Programs.git
```

2. Navigate to the project directory:

```bash
cd Python-Programs/Project\ 39\ Higher_Lower_Game_Using_Flask
```

3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

To start the Higher Lower Game, run the Flask application:

```bash
python app.py
```

Open your web browser and navigate to `http://127.0.0.1:5000` to start playing the game.

## How the Code is Made

The project is structured as follows:

- **Backend**: The game logic and server-side functionality are implemented using Flask, a lightweight web framework for Python.
- **Frontend**: The user interface is created using HTML, CSS, and JavaScript, providing a responsive and interactive gaming experience.
- **Routing**: Flask routes are used to manage the different pages and game states.
- **Error Handling**: The code includes error handling mechanisms to manage issues such as invalid inputs and server errors.
